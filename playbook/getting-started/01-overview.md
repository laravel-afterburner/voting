## Introduction

The Voting package lets your {{ entity_label }} run electronic ballots—for resolutions, elections, and other decisions—with configurable eligibility and optional proxy voting.

## Navigation

Open **Voting** from the main navigation. When proxy voting is enabled for your {{ entity_label }}, the menu also includes **Proxy votes**.

## Who can use Voting

Members who can **view ballots** for the current {{ entity_label }} see **Voting** in the main navigation and this playbook section. Proxy holders may access **Proxy votes** even when they cannot create ballots.
